Setup

  1. Download, unzip, and move the GitSpy folder into the root directory of your local {{orgName + '/' + repoName}} repository.
  2. Add "GitSpy/*" to your .gitignore file
  3. Replace the placeholder values in the KEY_USERS.js file with the ones below
      user_id: {{users_id}}
      dashboard_id: {{dashboards_id}}
  4. Make a commit to start tracking your local file changes on GitSpy
  5. Now you are setup with GitSpy <("<) (^"^) (>")>
