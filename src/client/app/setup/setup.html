Setup
Run these instructions once to start tracking your local activity on the {{orgName + '/' + repoName}} GitSpy dashboard. (If you have multiple GitSpy dashboards, you will need to run these instructions once per repo.)
1) Download, unzip, and move the gitSpy folder into the root directory of your local {{orgName + '/' + repoName}} repository
2) In your terminal, run the command `npm install --save fs` from your project's root folder.
3) In your terminal, run the command `node gitSpy/gitSpyInit.js` from your project's root folder.
This will add GitSpy post-commit hooks to your repo, enable a `spyUpdate` terminal command, and add gitSpy/* to your .gitignore file.
4) Replace the placeholder text in the SIGNATURE_HASH.js file with the one below. This signature hash will be different for each of your GitSpy dashboards:
    signature_hash: "{{signature_hash}}"
5) You are now set up with GitSpy! GitSpy will be updated any time you commit or run the `spyUpdate` command.
