Setup
1) Download, unzip, and move the gitSpy folder into the root directory of your local {{orgName + '/' + repoName}} repository
2) In your terminal, run the command "node gitSpy/gitSpyInit.js" from your project's root folder.
This will add GitSpy post-commit hooks to your repo.
3) Add "gitSpy/*" to your .gitignore file to avoid pushing your private GitSpy keys to GitHub
4) Replace the key in the USER_KEYS.js file with the one below
    signature_hash: "{{signature_hash}}"
5) Make a commit to start tracking your local file changes on GitSpy
6) You are now setup with GitSpy!
