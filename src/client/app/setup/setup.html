Setup
Run these instructions once to start tracking your local activity on the {{orgName + '/' + repoName}} GitSpy dashboard. (If you have multiple GitSpy dashboards, you will need to run these instructions once per repo.)
1) Download, unzip, and move the gitSpy folder into the root directory of your local {{orgName + '/' + repoName}} repository.
2) If you don't have node installed on your computer, run `brew install node` from your terminal.
3) In your terminal, run the command `sh gitSpy/gitSpyInit` from your project's root folder.
This will add GitSpy post-commit hooks to your repo, enable a `spyUpdate` terminal command, and add gitSpy/* to your project's .gitignore file.
4) Replace the placeholder text in `gitSpy/SIGNATURE_HASH.js` with the one below. If you have multiple GitSpy dashboards, this signature hash will be different for each repo:
    signature_hash: "{{signature_hash}}"
5) You are now set up with GitSpy! GitSpy will be updated any time you commit or run the `spyUpdate` command.
